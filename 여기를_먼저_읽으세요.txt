===========================================
   AI 프로젝트 대시보드 - 처음 사용자용
===========================================

[ 1단계 ] 실행하기 파일을 더블클릭하세요
------------------------------------------

   실행하기  (아이콘이 2개 보일 수 있습니다)
   ↑
   이 파일을 더블클릭!


[ 2단계 ] 잠시 기다리세요 (3-5초)
------------------------------------------

   "서버 시작 중..." 글자가 보이다가
   → 인터넷 브라우저가 자동으로 열립니다


[ 3단계 ] 대시보드 사용하기
------------------------------------------

   화면에 프로젝트 카드들이 보이면 성공!

   - 카드 클릭: 자세히 보기
   - 데이터 수정: 큰 보라색 "데이터 수정" 버튼 클릭
   - 종료: "종료하기" 파일 더블클릭


[ 4단계 ] AI로 데이터 수정하기 (NEW!)
------------------------------------------

   1. "데이터 수정" 버튼 클릭
      → Gemini 또는 ChatGPT 새 탭 자동 열림
      → 오른쪽에 JSON 에디터 표시

   2. Chrome 분할 보기 사용 (중요!)
      - 위쪽 새 탭(Gemini/GPT) 우클릭
      - "새 탭 그룹에 추가" 선택
      - 탭 그룹 우클릭 → "탭을 분할 보기에 추가"

   3. AI에게 요청
      예: "프로젝트 추가해줘 - 제목: 블로그 만들기"

   4. 수정된 JSON 전체 복사
      → 오른쪽 에디터에 붙여넣기
      → "저장" 버튼 클릭!

   5. 대시보드에 즉시 반영됨!


===========================================
자주 묻는 질문
===========================================

Q1. "실행하기"를 눌렀는데 아무 일도 안 일어나요
A1. 3-5초 정도 기다려주세요.
    브라우저가 자동으로 열립니다.

Q2. 브라우저가 안 열려요
A2. 수동으로 브라우저를 열고
    주소창에 입력: localhost:3000

Q3. 종료는 어떻게 하나요?
A3. "종료하기" 파일을 더블클릭하세요.
    "종료되었습니다" 메시지가 나옵니다.

Q4. 프로젝트 데이터를 바꾸고 싶어요
A4. "데이터수정" 파일을 더블클릭하세요.
    메모장이 열리면 수정 → 저장 → 브라우저 새로고침

Q5. 다른 컴퓨터에서도 사용할 수 있나요?
A5. 네! 이 폴더를 통째로 복사해서
    USB에 담아가거나 압축해서 보내도 됩니다.

Q6. 인터넷이 필요한가요?
A6. 아니요! 인터넷 없이도 작동합니다.
    (브라우저만 있으면 됩니다)


===========================================
파일 설명
===========================================

실행하기       → 대시보드 시작
종료하기       → 대시보드 종료
데이터수정     → 프로젝트 정보 편집
dashboard      → 프로그램 파일 (건드리지 마세요)
node           → 실행 파일 (건드리지 마세요)
